# Gossup

Gossup is a simplistic Go based system automation tool, which embeds the desired
state and test suite into a binary to be distributed to end-systems for
realization.

Inspired by [Goss][].

## Why use Gossup?

- Gossup is EASY!
- Gossup is FAST!
- Gossup is "SMALL!"
